Volunteers Server

A simple Express server exposing a volunteers API for reading and adding volunteer entries.

About this server

This server provides a small REST API used by the Foreign Collective project to store and retrieve volunteer records. It is built with Node + Express and exposes one primary resource: /api/volunteers/.

How to add volunteers

To add a volunteer, send a POST request to the volunteers endpoint with a JSON body. Example using curl:

curl -X POST /api/volunteers/ \
    -H "Content-Type: application/json" \
    -d '{"name":"Alex Chen","email":"alex@example.com","cause":"Park cleanup"}'

Server should respond with the created resource (201) or an error if the payload is invalid.

How to read volunteers

To read all volunteers, open the endpoint in your browser or fetch it programmatically:

/api/volunteers/

Example JSON response (array of volunteers):

[
    {
        "id": 1,
        "name": "Alex Chen",
        "email": "alex@example.com",
        "cause": "Park cleanup"
    }
]

Notes & troubleshooting

  • Make sure the server is running (npm start) before calling the endpoints.
  • Use proper Content-Type headers when POSTing JSON.
  • If you see a CORS error when calling from a different origin, enable CORS in the Express app.
